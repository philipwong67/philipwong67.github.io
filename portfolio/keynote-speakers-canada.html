Keynote Speakers
Canada

Keynote Speakers Canada is a Canadian speakers bureau representing Canadian Keynote Speakers since 2010.

The Insight

Keynote Speakers Canada has a very strong reputation over the years. As part of the growing network, there is a need to build a better web platform to showcase the Speaker Pool at Keynote Speaker Canada. This project is worked in association with Power Develoment Group.

The Idea

The Search Functionality

The main objective is to ensure users can find Speakers in the most intuitive way. With the new design, the Speakers is categorizes under different Speaker type and a prominent speaker search bar available on homepage along with key featured speakers that Keynote Speakers Canada would like to promote.

Keynote Speakers Search Bar to filter either by name or genre.

Speakes Details Page

The new Speaker profile page is also enriched with insightful information. It provides video placement, book promotions as well as Keynote topics expertise to better serve users to select their ideal Keynote Speaker.

Our focus is your audience.

Keynote Speakers Canada

The Result

A responsive design layout that provides clear information for users to navigation through the site in different devices. The Search Speaker system was a great intuitive feature that allows users to easily find available speakers under different categories or simply search by name.
